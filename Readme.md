## Featuring                

- Multi-Threading
- Anti Aliasing (Adaptive)
- Reflection
- Refraction
- Perspective / Orthographic Camera
- Blurry Refr./Refl.
- Planes, Triangles, Rectangles, Circles, Spheres
- Area Lights (Rect. / Circle)

- Obj. Mesh rendering
- GUI
- Caustics
- Radiosity / Global Illumination
